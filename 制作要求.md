# “四极·灵变”可重构机器人STEAM教育平台网站需求文档

## 一、 项目概述与定位

### 1.1 项目背景与背景

本项目是北京交通大学顶尖机构学实验室科研成果的商业化与教育化落地，旨在参加 **2026年“挑战杯”全国大学生课外学术科技作品竞赛/中国大学生创业计划竞赛**。

核心产品是基于“全R副三阶四面体”和“三棱柱”构型的**空间多环闭链可重构机器人**。项目秉承姚燕安教授提出的 **MSEA（Mathematics, Science, Engineering, Art）** 教育理念，采用**“硬件销售 + 深度课程赠送”**的商业模式，旨在将高阶机构学知识转化为青少年可感知的教育产品，填补高阶空间思维训练市场的空白。

### 1.2 网站核心目标

1.  **商业转化**：作为核心售卖门户，通过清晰的“产品+课程”包展示，直接驱动 B 端和 C 端购买。
2.  **品牌展示**：建立高科技、前沿、专业的“新工科”教育品牌形象，展示 MSEA 理念的独特性。
3.  **价值传递**：向“挑战杯”评委、投资人、客户清晰传达产品的技术壁垒（专利机构）与商业潜力。
4.  **低成本运维**：利用 **GitHub Pages** 实现全站静态化免费托管，确保极致的访问速度与零成本维护。

### 1.3 核心竞争力 (针对“挑战杯”优化)

- **技术壁垒**：源自北交大实验室专利，具备高变形比、全地形适应性等科研级性能。
- **教育闭环**：不只是卖玩具，而是提供一整套包含机械、电子、编程、艺术的 MSEA 课程体系。
- **竞赛赋能**：产品直接对接青少年科技创新赛事，为学校提供“硬件+培训+参赛”的一站式解决方案。

### 1.4 目标用户

- **B端客户**：中小学信息技术/科学教师、学校采购负责人、教育培训机构、青少年宫/科技馆。
- **C端客户**：关注孩子科创能力培养的初高中生家长、对机器人有浓厚兴趣的学生。
- **评委/投资人**：关注项目商业落地能力、技术先进性与社会价值。

### 1.5 商业模式 (优化点)

- **产品包销售**：购买硬件即送全套基础课程（PDF教材 + 视频演示）。
- **进阶增值服务**：提供针对“挑战杯”、“科创大赛”的专项导师指导服务。
- **校企合作**：为学校建立“变胞机器人创新实验室”，提供课程授权与耗材供应。

## 二、 网站整体风格与技术要求

### 2.1 视觉风格

- **主色调**：科技蓝、金属银、洁净白。
- **辅助色**：活力橙（用于按钮、高亮，代表创造与活力）。
- **风格关键词**：**前沿、专业、清晰、互动感**。避免过于低龄化的卡通风格。
- **必备视觉元素**：
  - 机器人**动态变形/运动的高清视频或GIF**（最具冲击力的素材）。
  - 北京交通大学实验室、姚燕安教授团队的相关影像/IP授权素材。
  - 课程教学场景（学生动手、编程界面）的真实或高质量渲染图。

### 2.2 技术要求

- **开发语言**：HTML5, CSS3, JavaScript (建议使用Vue.js/React等框架提升交互体验)。
- **部署平台**：**GitHub Pages** (要求全站静态化，确保在 GitHub Pages 环境下稳定运行)。
- **响应式设计**：必须完美适配PC、平板、手机等不同尺寸设备。
- **性能要求**：图片、视频需优化加载速度（利用 CDN 或 GitHub 仓库优化），确保页面流畅。
- **后台需求**：由于使用 GitHub Pages，建议采用**基于 Markdown 的轻量级内容管理**或静态网站生成器逻辑，便于通过 Git 直接更新产品信息。

## 三、 网站结构规划与页面详情

### 3.1 全局导航栏

固定于页面顶部，始终可见。

- **Logo**：“四极·灵变”或项目名称 + 小型图标。
- **菜单项**：[首页]、[产品中心]、[课程体系]、[科创竞赛]、[关于我们]、[新闻动态]、[立即咨询/购买]（突出显示按钮）。

### 3.2 首页

**定位：** 价值总览与吸引力入口。

- **首屏大图/视频**：全屏轮播展示机器人炫酷变形过程，叠加核心Slogan：“从搭积木，到造变形金刚”、“开启空间思维新维度”。
- **核心优势摘要**：用3-4个图标+短句展示：
  1.  **北交大顶尖科研转化**
  2.  **专利变胞机构，形态百变**
  3.  **PBL项目式阶梯课程**
  4.  **直通科创竞赛**
- **产品线快览**：展示“Geo-Lite（基础四面体）”和“Geo-Pro（三阶四面体/三棱柱）”的产品图、简要名称与定位，点击跳转至产品页。
- **应用场景展示**：图文展示“课堂教学”、“社团活动”、“家庭探索”、“竞赛现场”等场景。
- **新闻/动态滚动**：最新赛事成绩、合作新闻等。

### 3.3 产品中心页

**定位：** 详细展示硬件产品，促进购买决策。

- **产品分类导航**：清晰划分 **“基础探究系列”** 与 **“科研进阶系列”**。
- **每个产品独立展示区**：
  - **产品名称与型号**（如：Geo-Lite 基础版）
  - **高清多角度渲染图/实物图**
  - **核心参数表**：
    | 项目 | Geo-Lite (基础版) | Geo-Pro (三阶四面体) | Geo-Pro (三棱柱) |
    | :--- | :--- | :--- | :--- |
    | **适用学段** | 小学高年级-初中 | 初中-高中 | 初中-高中 |
    | **驱动舵机** | 6个标准数字舵机 | 8个总线舵机 | 9个总线舵机 |
    | **核心特点** | 安全易上手，趣味入门 | 全自由度，还原论文设计 | 冗余驱动，稳定性强 |
    | **编程方式** | Scratch/图形化，APP遥控 | Python/C++， 高级API | Python/C++， 高级API |
    | **预估售价** | ¥499 - ¥799 | ¥1，500 - ¥2，500 | ¥1，500 - ¥2,500 |
  - **技术亮点详解**：如“1.9倍高变形比”、“全地形适应性”、“模块化设计”。
  - **配套清单**：列出套件包含的所有部件（舵机、主控、结构件、电池、教程等）。
  - **“立即咨询/获取报价”按钮**（B端导向强）。

### 3.4 课程体系页

**定位：** 展示附赠课程的核心价值，深度诠释 **MSEA** 教育理念。

- **课程理念阐述**：从“平面拼搭”到“空间重构”，从“数学抽象”到“艺术表达”的 **MSEA** 理念图解。
- **四模块课程体系图**：用信息图展示四大模块及其递进关系：
  1.  **模块一：机械设计与空间几何 (Mathematics)**（培养空间构造思维）
  2.  **模块二：电子硬件与嵌入式控制 (Science)**（培养多机协同逻辑）
  3.  **模块三：运动规划与编程逻辑 (Engineering)**（培养计算思维）
  4.  **模块四：物理仿真与科研探究 (Art)**（培养科研创新能力）
- **各模块详情展开**：每个模块可点击展开，展示：
  - **教学目标**
  - **核心知识点**（如：SolidWorks建模、自由度分析、IMU姿态解算、步态规划）
  - **学生将完成的项目**（如：设计一个R副连杆、编写翻滚动作组）
- **课程配套资料展示**：《玩转立体几何》教材封面、教学PPT截图、学生作品集锦。

### 3.5 科创竞赛页

**定位：** 展示产品赋能竞赛的能力，吸引学校和尖子生。

- **赛事关联**：列出产品可直接支持的竞赛，如“全国青少年科技创新大赛”、“中小学生电脑制作活动”等。
- **“以赛促销”策略介绍**：宣传计划举办的“全国变胞机器人挑战赛”。
- **往届成果展示**：展示团队或早期试用学生获得的奖项、证书。
- **竞赛解决方案包**：提供针对学校竞赛培训的“竞赛套装”产品介绍。

### 3.6 关于我们页

**定位：** 建立信任，展示强大背景。

- **核心团队介绍**：重点突出 **姚燕安教授**（学术领袖）及**刘然副教授**（研发核心），附照片、头衔、成就（如MSEA理念提出者）。
- **科研溯源 (Challenge Cup Special)**：
  - **专利墙**：展示核心变胞机构、多环闭链结构等国家发明专利证书。
  - **论文成果**：列出在 ASME、Mechanism and Machine Theory 等国际顶尖期刊发表的相关学术论文。
  - **实验室背景**：展示北京交通大学机构学实验室环境、科研项目经费支持证明。
- **发展历程**：从科研成果到教育转化的关键节点。
- **合作伙伴**：已建立合作关系的学校、机构Logo墙。

### 3.7 页脚

- **联系方式**：邮箱、电话、微信公众号二维码。
- **办公地址**：北京交通大学相关地址。
- **友情链接**：北京交通大学官网、相关实验室主页。
- **版权信息**。

## 四、 核心功能点与交互需求

1.  **产品对比功能**：在产品页，允许用户勾选2-3款产品进行参数对比（弹出对比表格）。
2.  **资料下载区**：在“课程体系”或独立页面，提供产品说明书、课程大纲、教学案例等PDF下载（需留下联系方式后获取）。
3.  **询价/咨询表单**：在多个页面设置浮窗或入口，表单字段包括：姓名、单位/学校、职位、电话、需求描述（B端/C端/竞赛合作）。
4.  **动态内容展示**：在首页或新闻页，能够滚动展示最新的合作动态、比赛通知、教学视频。
